# Copilot Conversation Log

## Topic: Obsidian Vault Organization

Discussed approaches to organizing a large Obsidian vault:
1. Folder-based (PARA method)
2. Tag-based (flat vs hierarchical)
3. Link-based (MOCs, hubs)

## Decision
Go with hierarchical tags + minimal folders. Tags are more flexible and searchable.

## Follow-up
Build a tagging agent to automate the migration.
