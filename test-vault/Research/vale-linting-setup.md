# Vale Linting Setup Guide

## What is Vale?
A syntax-aware linter for prose. Enforces style rules on documentation.

## Installation
```bash
brew install vale
```

## Configuration
Create a .vale.ini in the repo root.

## Custom Rules
We should create rules for:
- Protocol-specific terminology
- Acronym definitions
- Code example formatting
