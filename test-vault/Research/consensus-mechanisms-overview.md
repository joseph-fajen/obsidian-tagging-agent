# Consensus Mechanisms Overview

## What is Consensus?
A consensus mechanism is a protocol that ensures all nodes in a distributed network agree on the current state of the blockchain.

## Types

### Proof of Work (PoW)
The original consensus mechanism used by Bitcoin. Miners compete to solve computational puzzles.

### Proof of Stake (PoS)
Validators are chosen to create blocks based on their staked tokens.

### Delegated Proof of Stake (DPoS)
Token holders vote for delegates who validate transactions.

## Key Tradeoffs
- Security vs. throughput
- Decentralization vs. efficiency
- Energy consumption vs. speed
