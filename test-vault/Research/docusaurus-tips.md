# Docusaurus Tips and Tricks

## MDX Components
Create reusable components for:
- Code examples with tabs (multi-language)
- API endpoint cards
- Version-specific content

## Sidebar Configuration
Use sidebars.js for custom ordering. Don't rely on filesystem ordering.

## Search
Algolia DocSearch is free for open source projects. Apply early — approval takes 2-3 weeks.

## Versioning
Only version when you have breaking changes. Too many versions = maintenance burden.
