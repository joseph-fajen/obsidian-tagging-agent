# Documentation as Code

## Core Idea
Treat documentation the same way you treat code: version control, CI/CD, linting, testing, review processes.

## Tools
- Docusaurus (React-based static site)
- Vale (prose linting)
- GitHub Actions (CI/CD)
- Playwright (screenshot testing)

## Benefits
- Single source of truth
- Automated freshness checks
- PR-based review workflow
- Versioned alongside code

## Our Adoption Plan
1. Migrate to Docusaurus (done)
2. Add Vale linting (in progress)
3. Set up CI/CD pipeline (planned)
