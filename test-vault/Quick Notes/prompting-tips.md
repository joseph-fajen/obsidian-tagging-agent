# Prompting Tips I've Learned

## XML Tags
Using XML tags like <context> and <instructions> helps Claude parse complex prompts.

## Few-Shot Examples
Always include 2-3 examples of desired output. #prompting is much more effective with examples.

## Chain of Thought
Ask Claude to "think step by step" for complex reasoning tasks.

## Negative Examples
Show what you DON'T want. This is surprisingly effective for #technical-writing tasks.
