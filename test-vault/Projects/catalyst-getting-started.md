# Getting Started Guide — Draft

## Prerequisites
- Node.js 18+
- A funded wallet (testnet)

## Step 1: Install the CLI
```bash
npm install -g @protocol/cli
```

## Step 2: Create a Wallet
```bash
protocol wallet create --network testnet
```

## Step 3: Your First Transaction
```bash
protocol transfer --to <address> --amount 1.0 --network testnet
```
